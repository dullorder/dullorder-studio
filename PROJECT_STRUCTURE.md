# Project Structure — DullOrder Studio

DullOrder Studio publishes multiple NFT art collections with distinct visual styles.

## Core Collections

DullOrder
2D stylized animal character art

CalmUnion
3D stylized character art

OmBrow
Realistic rendered animal art

## Creative Principles

• Original character systems  
• Trait-based variation  
• Animal diversity themes  
• Independent manual creation  
• Non-template visual development  

## Utility Model

• Holder verification via official email
• Bonus reward programs
• Upgrade and bonus NFT distribution
• No public community requirement model
